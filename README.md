# PHP Monitor

**PHP Monitor** (phpmon) is a free, open-source macOS utility that lives in your menu bar and gives you instant visibility and control over your PHP environment.

## Repositories

| Repository | Description |
|---|---|
| [`phpmon`](https://github.com/nicoverbruggen/phpmon) | The main application, which is not part of this organization |
| [`early-access`](https://github.com/phpmon/early-access) | Early access for the main application; gated behind GitHub sponsorship |
| [`website`](https://github.com/phpmon/website) | Source code for the website, which is made with Laravel (phpmon.app) |

Some other repositories in this organisation are currently private, like the API, for example. Those may still end up being open sourced in the future.

## Open Source & Free

PHP Monitor is entirely free to use — there are no paid tiers, no feature paywalls, and no telemetry collected by default. The project is sustained by community donations and sponsorships.

If the app saves you time, consider [sponsoring the project](https://github.com/sponsors/nicoverbruggen) or starring the repository. Both go a long way toward keeping the app actively maintained.
